# Sales Performance Analysis

This project analyzes structured sales data using SQL for querying, Python (Pandas, Matplotlib) for data cleaning and exploratory analysis, and Power BI for interactive dashboard visualization. The analysis identifies revenue trends, profit margins, customer segmentation, and top-performing products to help businesses make data-driven strategic decisions.

# Dataset
Dataset Information:<br>
Total Records: 200<br>
Total Columns: 14 <br>
Missing Values: 0 <br>

Data Includes:<br>
Sales transactions<br>
Product details<br>
Revenue and cost metrics<br>
Customer information<br>
Regional data<br>
Time-based sales trends<br>
