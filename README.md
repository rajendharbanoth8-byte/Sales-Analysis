# Sales Performance Analysis

This project analyzes structured sales data using SQL for querying, Python (Pandas, Matplotlib) for data cleaning and exploratory analysis, and Power BI for interactive dashboard visualization. The analysis identifies revenue trends, profit margins, customer segmentation, and top-performing products to help businesses make data-driven strategic decisions.

#  Key Insights

* Electronics generated the highest profit contribution.
* Laptop, Desk, and Printer were the top-performing products.
* No loss-making products were identified.
* Sales and profit showed variation, indicating that higher sales do not always result in higher profit.
* Regional performance differed across locations, creating opportunities for targeted sales strategies.
* February and May recorded peak sales activity.
* Pricing and profit margin analysis highlighted opportunities to improve profitability.
